= AdocForge Roadmap
:toc:

== Phase 0: Repository Foundation

* [x] pnpm workspace and strict TypeScript configuration
* [x] ESLint, Prettier, Vitest, Playwright, Changesets, and CI configuration
* [x] Three public package entry points and a buildable playground
* [x] Architecture document and ADR 0001 through 0004
* [x] Community and security policy documents

== Phase 1: Core Conversion

* [x] Wrap Asciidoctor.js 4 behind `AsciiDocProcessor`.
* [x] Return HTML, outline, title, and structured diagnostics.
* [x] Enforce safe defaults and test disabled dangerous features.

== Phase 2: Editor and Preview

* [x] Integrate CodeMirror 6 without exposing `EditorView` publicly.
* [x] Add debounced preview and outline updates.
* [x] Define the stable Web Component properties, methods, and events.

== Phase 3: Local Persistence

* [x] Implement the IndexedDB storage adapter.
* [x] Add autosave state and reload recovery.
* [ ] Add `.adoc` import and export.

== Phase 4: AI Assistance

* Implement rewrite, summarize, and continue operations.
* Support cancellation and optional streaming.
* Require explicit Accept or Reject for generated changes.

== Phase 5: Public Release

* Complete API documentation, examples, accessibility checks, package inspection, and release automation.
* Publish only after explicit approval.
